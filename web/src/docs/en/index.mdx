## What is Naty?
Naty is the effortless starter for every project.

You can create project easily with `npm create naty@latest`

<br />
---
<br />

## Quick Start
1. Run `npm create naty@latest`
2. Configure project settings
3. Run `cd "Project Name"` to move to the project directory.
4. Run `npm run dev`
